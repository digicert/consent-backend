***** Consent backend application *****
This is the backend application for the consent management system. It is a RESTful API that provides endpoints for managing consents.

Gradle multi-module project
This project is a multi-module project that uses Gradle as the build tool. The project is divided into the following modules:
client-consent
This module contains the client-facing API for managing consents. It provides endpoints for creating, updating, and deleting consents.
